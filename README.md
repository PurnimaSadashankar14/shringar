# Shringar – Makeup Artist Appointment Booking System

## Overview
Shringar is a web-based appointment booking platform designed to simplify makeup service scheduling between clients and makeup artists.

## Features
- Appointment Booking
- Admin Dashboard
- Client Management
- Service Management
- Appointment Tracking
- Responsive UI

## Technologies Used
- Laravel
- PHP
- MySQL
- Tailwind CSS
- JavaScript

## Installation
1. Clone the repository
2. Run composer install
3. Configure .env
4. Run migrations
5. Start local server

## Future Improvements
- Online payment integration
- Email notifications
- Real-time appointment status
